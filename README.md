## The MongoDB Shell

This repository is a monorepo for all the various components in the MongoDB Shell across
all environments (REPL, Browser, Compass, etc).

## Requirements

- NodeJS `~10.2.1`

## Install

```shell
npm install -g lerna
npm install
```

## Running Tests

```shell
lerna run test
```
